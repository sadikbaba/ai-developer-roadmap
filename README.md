## Stage 1 — Foundations
⏱ *4–6 weeks*

- **Python:** loops, functions, OOP, NumPy, Pandas
- **Math:** linear algebra, statistics, gradient intuition
- **Tools:** VS Code, Jupyter Notebooks, Git/GitHub

---

## Stage 2 — Machine Learning
⏱ *6–8 weeks*

- Supervised / unsupervised / reinforcement learning
- Algorithms: regression, decision trees, random forests, SVMs
- **Tool:** Scikit-learn
- Build **3–5 mini projects** (price predictor, spam classifier, etc.)

---

## Stage 3 — Deep Learning
⏱ *6–8 weeks*

- **Framework:** PyTorch (industry standard)
- **Architectures:** CNNs → RNNs → Transformers
- **Projects:** image classifier, sentiment analyser, text summariser

---

## Stage 4 — LLMs & Generative AI
⏱ *4–6 weeks*

- Prompt engineering, RAG, fine-tuning, AI agents
- **APIs:** OpenAI, Anthropic (Claude), Hugging Face
- **Tools:** LangChain, LlamaIndex, vector DBs (Pinecone, Chroma)
- **Projects:** custom chatbot on your own docs, AI news agent

---

## Stage 5 — Deployment & MLOps
⏱ *3–4 weeks*

- Serve models with **FastAPI** or Flask
- Containerise with **Docker**
- Deploy to AWS SageMaker / Google Vertex AI / Hugging Face Spaces
- Track experiments with **MLflow** or Weights & Biases

---

## Stage 6 — Pick Your Lane

| Track | Focus |
|---|---|
| 🧠 **LLM Engineer** | RAG, agents, fine-tuning, prompt systems |
| 👁 **Computer Vision** | Object detection, video AI, diffusion models |
| ⚡ **AI Product Builder** | Rapid API prototyping, no-code + code hybrid |
| 🔗 **AI for Web3** | Onchain data, DeFi bots, smart contract tools |

---

## Stage 7 — Portfolio & Market Entry

- ✅ 3–5 strong GitHub projects with clean READMEs
- ✅ Live demo site or deployed AI app
- ✅ Build in public on X (Twitter)
- ✅ **Freelance:** Upwork, Toptal, Contra
- ✅ **Full-time:** search "LLM engineer / AI engineer / ML engineer"
